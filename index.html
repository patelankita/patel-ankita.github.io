Flight Prediction

The assignment aims to predict two-hop flights that minimize the chance of missing a connection, given historical airplane on time performance data. Connections must have at least 45 minutes between landing and takeoff, and no more than 12 hours.

Flight On-Time Performance Analysis

The assignment aims to explore the world of airline on-time performance data analysis. The task is to implement a pipeline that runs a set of pseudo distributed map reduce tasks to plot the mean delay of the five most active airlines and for the five most active airports in the country.

Million Song Clustering

The assignment aims to perform iterative computations, learning clustering algorithms. Using Spark, performed hierarchical agglomerative clustering and k-means clustering to cluster songs based on length, hotness, tempo, loudness, etc.

Web App Maker

Web App Maker allows users to create and use online, mobile friendly, Websites. Users can create Websites. There are two types of users. Those who can create Websites as developers and end users who visit the Websites created by the developers. Only developers can create and modify the Websites. End users can not modify the Websites, they can only interact with them. Websites can contain multiple pages which in turn contain various kinds of widgets.
